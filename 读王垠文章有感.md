#### 原因与证明
```
    原因往往比证明来得更加简单，更加深刻，但却更难发现。对于一个事实往往有多种多样的证明，然而导致这个事实的原因却往往
只有一个。如果你只知道证明却不知道原因，那你往往就被囚禁于别人制造的理论里面，无法自拔。你能证明一个事物具有某种特性，
然而你却没有能力改变它。你无法对它加入新的，好的特性，也无法去掉一个不好的特性。你也无法发明新的理论。有能力发明新的事
物和理论的人，他们往往不仅知道“证明”，而且知道“原因”。

   为了证明而证明，其实就跟我们现实生活中的中国高考应试教育，没什么区别，活生生硬塞给你一些原理原则理论知识，而缺少最根本
的教育启蒙，给我们灌输的就是，我们生下来这些事物就是这样存在的，不要问我十万个为什么，因为爸妈和老师他们也不知道，所以我
们的好奇心和对思维模式的培养，没有得到很好的发展，我也不例外，很多时候没有带问题带着“为什么”的心态去学习及工作，为什么要
这样？出现这个的原因是什么，为什么需要和某个事物搭配才能出现？需要有这种思维的人们，还是得长期用这种方式去思考问题，长期
坚持，也就变为习惯了。

```
#### 论对东西的崇拜
```
    当一个人没有办法控制一个东西的时候，他就会对它产生“崇拜”的心理，这就像所有的宗教和迷信一样。当人们处于自然灾害的凌
威之下，没有能力掌握自己命运的时候，他们就对神和超自然的力量产生了崇拜。这是一种心灵的慰藉，至少有上帝或者观音菩萨，可以
聆听他们的心声，可以给予他们度过灾难的勇气，但它同时也显示出人的无助和自卑。这种无助和自卑，也引发了偏激的宗教心理，因为
他们害怕自己的“保护神”被别人的“保护神”所压倒，以至于让自己受制于他人。这是一种愚昧和卑劣的感情。
   
   我很喜欢看王垠的文章，他写的很多都是透过问题看本质，“崇拜”的心理几乎每个人都会有，比如我们还很小的时候，会崇拜大哥哥
做的玩具很厉害，很好玩；会崇拜动画片里面的动画人物技能有多厉害，小伙伴之间还经常为之吵架、打架；还有就是在很多国家都会
有的鬼神论，当自己或亲人受难时，总会求助于神的保佑，驱逐妖魔鬼怪，他们都是人们心中神圣不可侵犯的。偏激的宗教崇拜心理，
比如：最近看到朋友圈疯传的截图“骑士和勇士对骂群”，此事还引起腾讯的官方关注，对这些群号实行封号处理。其实有喜欢崇拜的东西
不是说不好，心中有信仰也是一件好事，但是偏激的过渡崇拜，会造成心理上的扭曲，容不得别人叽歪一句，否则就怼回去了。

    我对于很多事情都是保持平常心，对于偏激的崇拜心态，我觉得像是一种精神的鸦片一样，信仰成为生活的大部分，得到的只是心灵
的慰藉，真的是一种愚昧的感情；而这些并不能给自己带来改变，那有什么意义呢？

```
#### “解决问题”与“消灭问题”
```
爱因斯坦说“想象力比知识更重要”，也许就是这个道理。没有想象力的人经常钻牛角尖，走死胡同，忘记了自己其实还有另外的路可走。

在解决问题之前，我们应该先问自己三个问题：
这问题是否真的“存在”？
如果解决了这个问题，会给我和他人带来什么实际的好处？
这问题是否可以在简单的改变某些“设计”或者“思路”之后，不复存在？
```

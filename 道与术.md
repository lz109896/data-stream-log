## 《道与术》
### 追溯到一个寓言
```
   原文：“ 授人以鱼，不如授之以渔，授人以鱼只救一时之急，授人以渔则可解一生之需。”----《授之以鱼不如授之以渔》

   基本解释：
   这段话是出自道家，但至于出自谁手，至今学术界仍在考证当中。“授人以鱼，不如授之以渔”，说的是"送给别人一条鱼能解他
一时之饥，却不能解长久之饥，如果想让他永远有鱼吃，不如教会他捕鱼的方法"，有鱼吃是目的，会钓鱼是 手段。这句话说明。
要想帮助他人解决难题。还不如传授给他人解决难题的方法。
```
### 为什么写《道与术》？
```
   这是我好几次跟锋哥交流出现的次数最为较多的词语，在今晚与锋哥深刻交流学习中出现一个问题，自己的一个执行问题；

   我在多次交流中提到英语不好的，后面他给了实质性的学习方法；比如：http://lixiaolai.com/2016/06/11/makecs-appendix01/ 
和 http://lixiaolai.com/2016/06/12/makecs-preface/ ，这两篇文章都是李笑来老师的原文，都是最优质的资料昨天锋哥发给我，
我没细看，略读一番，文中的写的确实有道理，会在后面的文章介绍。写这篇文章的主要目的是总结这次的深刻交流后的输出，
因为锋哥给的学习方法、文章我没有学习后得出什么指导性方法！实质性计划！这是我觉得致命的。为什么致命的？
```

### 一个人的加速度
```
同样都是8K 起薪就业,表面都是8K,比如说：其实这两类人后面的发展轨迹完全不同
末速度=初速度+加速度乘以时间,初速度差不太多的,时间每个人一样的
加速度,每个人就差天各地了

加速度是什么？加速度在一个身上来说，就是对编程思想的一种认知，一种思维模式，当你和别人在同等的时间，同等的
条件环境下一起做一件事时，你有不同于其他人的编程思想和思考，这就是思维模式，这会成为你加速度的助推，持之以恒会让你
成为佼佼者

```

## "斜杆青年"一词
"斜杆青年"：（拥有多重职业和身份的多元生活的人群）
```
    斜杠青年来源于英文Slash，出自《纽约时报》专栏作家麦瑞克·阿尔伯撰写的书籍《双重职业》，指的是一群不再满足“专一职业”
的生活方式，而选择拥有多重职业和身份的多元生活的人群。这些人在自我介绍中会用斜杠来区分。

    例如，张三，记者/演员/摄影师，“斜杠”便成了他们的代名词。 斜杠青年越来越流行，已成为年轻人热衷的生活方式。

    "斜杆青年"一词，最早出现在我的世界，是在2017年热播的电视剧《恋爱先生》中，靳东饰演的男主角程皓，向女主角介绍自己
是：斜杠青年。就像程皓，既是牙医，又是恋爱顾问，拥有双重职业。在现在互联网大势下，不要为已经错过的惋惜，你只看到两年前
公众号刚起步的时候吸粉是多么容易，可是为什么你会错过呢，因为没有准备。微博、知乎、公众号……都是风口。错过了这次机会，还
会有下次下下次。但如果不行动，那你将终身错过。

    也不要轻易的以为做一个斜杠青年是多么的容易，首先必须理解“斜杠青年”这一个词的意思，在我查阅百度时，或者我们的日常生活
中，同时也包括我自己，发现似乎一个人只要身兼数职，就可以变成别人口里的 “斜杠青年”，走上令人仰慕的人生巅峰。然而其真正的内
涵并不是如此。

    就如拿教我开发编程的:锋哥，首先他是一家公司的管理者，有着别人为之羡慕的工作，年纪轻轻让人嫉妒，嫉妒是没用的，因为你
不知道他从中付出了多大的努力；重本专业毕业英语6级裸考。在这些坚实的基础下，他同时也是一个斜杠青年，是一个负责任的编程开发
职业导师，虽然他没有这么定义自己，但当我理解“斜杠青年”时，或许明白他推荐这本书我看的原因了。

    帮助需要被帮助的人，以帮助编程学习、职业转型为导向，就业为主，后续持续指导往上走，帮助别人，持续有偿的辅导 ———— 这是
他跟我说过的一句话。帮助这个词语，从中看出他对这份辅导工作是兴趣上的一种才华上的展现，展现人格魅力的一种方式，得到肯定，
有偿的辅导。有偿跟工资有很大的区别；
    
    “有偿”在百度百科意思是：民法上的有偿，是指双方存在利益上的交换事实，并非指双方所获利益的绝对相等。同时，民事法律行为
的有偿可表现为支付金钱，也可表现为提供技术、劳务或服务等。“工资（Wages）”是指雇主或者法定用人单位依据法律规定、或行业规
定、或根据与员工之间的约定，以货币形式对员工的劳动所支付的报酬。从字面上理解有偿更多的是双方之间的一种约定，相互之间的
交换，以及肯定，表现为不那么在乎利益，从而达成交易。工资是一种早已制度约束好的交易，支付员工的劳动报酬。

    有人认为有偿，谈何帮助，很多人潜意识里，只有无私的奉献才是对他人的帮助；呵呵！别人的时间不是时间，花费时间是对别人
的帮助其实也是人生的一种成本。
```
## 为什么是“斜杠/” 而不是其他连字符？
```
    可能很多人都会用斜杠做自我介绍：
有正经的：雕刻家/建筑师/音乐家/数学家/工程师/发明家/解剖学家/地质学家/制图师/植物学家/...
也有无厘头的：粽子爱好者/护花使者/LOL勇士/XX成功者/贯彻不吃葱姜蒜主义者/.....

    那么你呢，可以用斜杠介绍下自己吗?

    “斜杠青年” 并不是多打几份工，多做几份兼职的道理；在通往 “斜杠青年” 的路上，有的人只是东学一点，西学一下，
看似在追求多元的精彩生活，实则是在盲目肤浅地想通过急速变现获得成功。在这个竞争日益激烈的时代，真正的“斜杠青年”
才明白， 必须在一个领域做到优秀之后，才有可能获得真正的话语权。记得《斜杠青年》的作者 Susan 谈起过自己的离职，
那时候，很多人问她会不会很担心很害怕，会不会没有安全感？她说，当时的确会有一些不安全感，但是这种不安全感并不是
非常强，因为她知道自己有一个保底的技能，那就是她的英文能力。“斜杠青年” 最大的意义，就在于他们能够在多个维度上
发展自己，不断精进。而当我们在某个领域里做到了突出，然后再结合其他领域进行跨界，这就是从平庸走向卓越的最佳策略
    
    非常赞同 Susan 对斜杠青年的理解，通俗一点讲就是能“一专多能”甚至“多专多能”。有人会拿工匠精神出来怼斜杠，这其实
是两种不同人生的选择：专注于一个方向，做到极致，可敬；多面发展的，能跨界融合，可赞。我佩服那些一生只需要精通一个
行业的人，一招鲜啊；但我更赞赏那些能一理通百事的人，这样的人生更为丰富多彩。

得到App专栏《通往财富自由之路》作者;怎一个牛逼了得！
```

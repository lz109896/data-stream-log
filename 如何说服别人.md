#### 我自己其实是一个不太会说服的人，所以只能分享一些错误做法作为反面教材:
```
1、说服别人不是谁有道理听谁的:我总是喜欢用逻辑和客观事实来说服别人，这套东西放在工作上还行，但在与家人的沟通中就不太管用了，
说服家人更多的还是要用情感去感染对方。
2、说服态度很重要:有时觉得自己说的有理，就不知不觉趾高气昂了，但殊不知越是态度傲慢，对方越是不接受，因为人总是拒绝承认自己
比别人差。
3、说服不是替别人做决定:说服是要有一个度的，有时别人已经认可了你的观点，但你却还要替别人做下一步行动的决定，反而会招致别人
的反感，要知道人天生喜欢控制，不喜欢被控制。
```
#### 角度，关注点
```
说服从语言表达，从利益获取，从情感共鸣，从社会发展的角度，让说服对象自觉的产生认同，从而改变行为， 晓之以理，动之以情，往往
很多时候沟通的目的并不是为了说服别人，也不是为了说服自己，而是寻找共同的目标，合力去推进某件事情或达成某种默契，从某种意义说，
不是征服与被征服，不是零和博弈，而是共赢。 说服一个人，在于能够站在对方的思维角度，先肯定他某一方面的点，然后在逐步引导和影
响他的思维逻辑以及态度。重要的是——以诚相待！这些心理、生理上的说服技术，很大一部分成果靠的是一个封闭空间，只要脱离了物理空
间的限制，接触到真实环境，人们就很容易摆脱对方的说服力，这也说明为什么每次谈判都必须搞闭门会议，因为假如可以随便走动，变数
太大了，得不到任何谈判结果。让别人赞同自己的观点，就要抓住别人的敏感点，这种敏感点理所当然的是别人最关注的，最需要的，当然
这些既有人人相同的地方，比如健康，安全，经济，欲望等等，也因人而异，比如失眠的人就关注睡眠问题，身材不好的人就关注减肥，想
要说服人，首先要深度了解人，这样才能精准投放语言。站在对方的立场考虑问题分析问题，有些话点到为止，大家观点不一致时特别容易引
起情绪波动(一点要处理好情绪)，如果对方坚持自己想法，不防让对方先讲出来，你再分析对方的出发点，找出共同点，慢慢引导切勿心急。
```
#### 如何让别人接受你的批评—春风化雨的训人之道
```
如何让高高在上的训斥，让别人变得比较容易接受；没有人喜欢训斥，一般的正常人们也不会把训斥当成爱好，这是你作为前辈，或者阅历
相对丰富，也更有大局观，更容易发现后悔，或者下属在走一条已经被验明是错误的道路，这个时候，当然是出问题了，否则就是没有责任
心了，很多人只要觉得自己是对的，别人是错的，就会不由自主的高高在上，在别人耳朵里听起来就会很刺耳，到头来别人不接受你也不痛
快，那么作为长辈，头儿应该怎么让你的训斥变得不那么像训斥，从而比较容易让人接受呢？
```
#### 我建议咱可以分三步走：
```
1、以平辈自居而非前辈或长辈（身份的权威性反而成为说服的绊脚石，要消除身份对立感，奠定交流的感情基调） 
2、理解对方真实的动因，以同理心来建立信任感（谈话目的不是居高临下的训导） 
3、用自己的错误来说明真正正确的做法到底是什么？（本人也有类似的错误，拉近距离，使建议更容易被接受） 
```
#### 反面例子:中国国家博物馆的馆长陈履生
```
我举一个反面例子和一个正面的例子，好了，几个月前，中国国家博物馆的馆长陈履生在一档谈话类节目中教育大家不要在博物馆拍照，陈
馆长说，我觉得在博物馆拍照的人都很傻，走进博物馆应该多花时间去感受感受，文化与历史，不要总是惦记着拍照，你拍的那个像素也好，
清晰度也好，都不如我们官网上的图片，你去卢浮宫，看看蒙娜丽莎，你傻傻的拍啊。官网上的高清图片，连每一条裂文都能看得清清楚楚，
你下载就好了，你拍它干什么呢？节目放出来之后引起了很多人的反弹，有人说你凭什么高高在上的教育我呢？博物馆是一个为人提供知识
服务的地方，不是一个强制教育的场所，你是个当馆长的，应该多一点服务意识，少一些训导主任的味道，更尴尬的是，有人搜出了陈馆长
自己在国外的博物馆，拿着单反相机拍这拍那的照片儿，照片发在网上，下面一片网民就排着队，还说别人拍照傻，这回你自己打脸了吧，
其实陈馆长挺冤的，他并没有禁止大家在博物馆拍照的意思，他还力推让年轻人在博物馆举行婚礼，让博物馆成为大众化的休闲场所了，可
是他的那段话呀，却起了完全的反效果，原因就在于，第一它的姿态高高在上，首先在身份上就能逛博物馆，立敌受众对立了起来。

其实大家都知道，高清图片收起来很容易，可是大家还是要拍，是因为这基本上就是咱们现代人的生活方式了，凯撒有句名言叫做：我来了，
我看了，我征服了；而现代人的生活呢，就是：我来了，我拍了，我发朋友圈了，没拍照基本等于没来过，没发朋友圈基本没拍过；你懂吗？
健康和优雅的生活方式，可是你不理解这一点啊，简直没有办法跟现代人沟通；

第三种最严重的就是他自己犯了同样的错误，他自己也拍照，而且此后他一直没有对此进行回应，这就让他之前的训导觉得毫无说服力，甚至
自己，也沦为了笑柄，上一代的人在说服他人的时候，喜欢营造权威感，因为这样比较令人幸福，但是今天这个时代。身份上的权威性反而
会成为说服过程中的绊脚石，美国文化人类学家彼得将近50年前就写一本书叫做文化与承诺，一项有关代沟问题的研究，过去晚辈的知识和
技能完全通过长辈的传授获得，而现在随着新技术革命的发展，年轻一代。还能不断的将他们对社会生活的理解和自己创造的新知识，传递
给他们的前辈，这种文化反哺的现象，意味着长辈的权威已经逐渐不在了，然而很多时候就找不到这个变化，陈馆长的例子就是一个典型的

#### 正面例子:傅雷家书
```
那正确的做法应该是怎样的呢？我再举一个正面的例子，在傅雷家书中，有一篇傅雷先生写给儿子的信，他是这么写的，长篇累牍的给你写
信啊，不是空唠叨，不是莫名其妙的闲扯，而是有好几种作用的，第一个，把你当做一个讨论艺术，讨论音乐的对手；第二我想激出你一些
感想，让我这个做父亲的也得些新鲜的养料；第三个通信训练不但是文笔，尤其是思想；第四，我想时时刻刻随处给你做一个警钟，做一个
忠实的镜子。你看，他以平辈和对手的身份说话，消除身份上的对立感，也奠定了对话基调，第二点，虽然提及了他做父亲的身份，但目的
也是请教，而不是居高临下的训导，而且他带出了我也会有问题，我也需要你帮助的意思，其目的是为了告诉儿子，我这个做父亲的都需要
新鲜养料来不断提升自己，更何况你这个尚且年轻的儿子呢，前面的是你帮我，那现在顺理成章，也要允许我帮你训练训练你喽，第四点说

沟通方式。

你可以想象一下，如果陈馆长也像傅雷先生这样说话，会不会觉得好接受很多？如果我是馆长的话，我会干脆接着这么说：自己也喜欢在博
